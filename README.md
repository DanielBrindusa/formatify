# Image ⇄ ICO Converter

A simple client-side converter for GitHub Pages.

## Features

- Convert PNG, JPG, JPEG, and ICO
- Export to PNG, JPG, JPEG, or ICO
- Optional one-click export for all formats
- Runs fully in the browser
- No backend required

## How to use on GitHub

1. Upload all files from this folder to a GitHub repository.
2. Enable GitHub Pages for the repository.
3. Open the published page.
4. Upload an image and convert it.

## Files

- `index.html`
- `style.css`
- `script.js`

## Notes

- JPG/JPEG do not support transparency.
- ICO export is created as a single-image `.ico` file.
